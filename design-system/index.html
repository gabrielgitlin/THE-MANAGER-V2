Logo Usage Brand guidelines 2025 — Andrés Higueros
Primary: icon + wordmark lockup · white on black
The Three Brand Assets
Icon — white on black
Icon — black on white
Wordmark — white on black
Wordmark — black on white

Lockup — black on white
Approved Background Colors
#000000 — Black
#0c0c0c — App BG

Palette Cream
Do not place on photography, brand green, or any non-approved color. Black and white are always safe.
Minimum Sizes
Wordmark: 78px digital · 26mm print
Icon: 60px digital · 20mm print
Lockup: 80px digital · 35mm print
Clear Space
Minimum clear space = height of "M" in logotype on all sides.
For icon = ½ icon height on all sides.
For lockup = height of icon on all sides.
App Usage
Nav header: wordmark only · filter:invert(1) on #111111
Favicon / app icon: TM monogram only
Loading / splash: lockup centered on black
Incorrect Uses — Never Do
✕ Stretch, squish, or distort any element
✕ Rotate the logotype or icon
✕ Apply shadows, glows, outlines, or gradients
✕ Use in brand green or any non-approved color
✕ Place on busy photography or dark imagery
✕ Use low-resolution or rasterized at small sizes
✕ Change the typography or rearrange logotype elements
✕ Separate icon from wordmark in the lockup
✕ Use the icon alone where the lockup fits

Folder Containers Top-left radius only — analog filing system reference, used selectively
The folder container pattern applies a 4px top-left radius only — all other corners stay square. Optionally, the right and/or bottom border is removed so the tab flows into its content. This references manila folders, file tabs, and archival labeling systems from The Manager's editorial DNA. Use sparingly. This is a deliberate accent, not a default card style.

Layout Rules
Order: Header → KPIs → Charts → Table — user scans top to bottom, big to small

Chart row: 2fr 1fr split — primary chart left, breakdown/summary right
Table: always last — most detailed, least skimmable — give it a header with count + action button
Spacing between sections: 16px inside dashboard · Page padding: 20px
